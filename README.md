* [ReactJS](https://react.dev/) - The library for web and native user interfaces

* [GraphQL](https://graphql.org/) - A query language for your API.

## Setup
To run this project, install it locally using [NPM](https://www.npmjs.com/):
For back-end
```
$ cd backend/
$ npm install
$ npm run:start
```
For front-end
```
$ cd frontend/
$ npm install
$ npm run start
```
